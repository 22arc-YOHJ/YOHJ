# YOHJ — Site Premium GitHub Pages

Ce dossier est prêt à être publié sur GitHub Pages.

## Publication
1. Décompressez le ZIP.
2. Dans le dépôt GitHub `YOHJ`, choisissez **Add file > Upload files**.
3. Importez directement `index.html`, `styles.css`, `script.js` et le dossier `assets`.
4. Validez avec **Commit changes**.
5. Allez dans **Settings > Pages**.
6. Choisissez `main` et `/(root)`, puis cliquez sur **Save**.
